# llm_4_word_vba

VBA-макросы, добавляющие функционал работы с LLM (Large Language Models) в MS Word.

## Возможности

- **Одиночный запрос** — отправка промпта (и, опционально, выделенного текста) в LLM через OpenAI-совместимый API; результат заменяет выделенный текст или возвращается как ответ.
- **Чат-интерфейс** — пользовательская форма с окном чата, позволяющая вести беседу с LLM и также вставлять результаты в текст документа.
- **System prompt** — настройка системного промпта через реестр Windows (HKCU).
- **Настройки в реестре** — URL API, ID модели, API-ключ и системный промпт хранятся в `HKCU\Software\LLMWordMacro\`.

## Структура проекта

```
doc.dotm          — шаблон/надстройка Word с встроенными макросами
vba/
├── LLM_work.bas  — основной модуль (логика запросов, реестр, парсинг JSON)
├── LLM_chat.frm  — пользовательская форма чата (текстовое поле, кнопки)
└── LLM_chat.frx  — бинарные данные формы (ресурсы, не редактируются)
```

## Запуск

Откройте файл **doc.dotm** в MS Word. Приложение само создаст необходимый файл с подключенными макросами.



## Развертывание макросов в новом документе

Если необходимо создать новый документ:

1. Запустите приложение MS Word.
2. Откройте редактор VBA (**Alt + F11**).
3. В меню редактора VBA выберите **Файл → Импорт** и импортируйте:
   - `vba\LLM_work.bas`
   - `vba\LLM_chat.frm`
4. Сохраните документ/надстройку.


## Настройка разрешений VBA Word (при необходимости)

Убедитесь, что макросы не заблокированы:

1. **Файл → Параметры → Trust Center → Центр управления безопасностью → Параметры макросов**.
2. Выберите **Отключить все макросы с уведомлением** (или **Включить все макросы** для тестирования).


## Конфигурация LLM

1. В Word запустите макрос **ConfigureLLMSettings** (через **Разработчик → Макросы** или **Alt + F8**).
2. Заполните поля:
   - **LLM_API_URL** — URL API (например, `https://api.openai.com/v1/chat/completions`).
   - **LLM_MODEL_ID** — идентификатор модели (например, `gpt-5.1`).
   - **LLM_API_KEY** — Bearer API-ключ.
   - **LLM_SYSTEM_PROMPT** — системный промпт (необязательно).
3. Настройки сохраняются в реестр Windows: `HKCU\Software\LLMWordMacro\`.


## Использование

- **Одиночный запрос:** запустите макрос `RunLLMQuery`, введите промпт (и выделите текст в документе для его включения в запрос). Результат заменит выделенный текст.
- **Чат:** запустите макрос `RunLLMChat` для открытия окна чата. Введите сообщение в нижнее поле и нажмите **Отправить**. Ответ вставится в историю чата; кнопка **Вставить последний ответ** помещает последний ответ в документ (заменяет выделенное содержимое).


## Настройка ленты

Чтобы добавить кнопки макросов на ленту Word:

1. **Файл → Параметры → Настроить ленту**.
2. В правой колонке (Настроить ленту) нажмите **Создать вкладку** → переименуйте вкладку в **LLM** (правой кнопкой → Rename).
3. В той же вкладке нажмите **Создать группу** → переименуйте группу в **LLM Макросы** (или иначе).
4. В левой колонке (Выбрать команды) выберите **Макросы** из выпадающего списка.
5. Найдите макросы `LLM_work.RunLLMQuery`, `LLM_work.RunLLMChat`, `LLM_work.ConfigureLLMSettings` и нажимайте **Добавить >>** для переноса в группу «LLM Macros».
6. Для каждой кнопки можно изменить значок: выделите кнопку → **Переименовать** → выберите иконку и подпись.
7. Нажмите **OK** для сохранения.

После настройки на ленте появится вкладка **LLM** с кнопками для быстрого доступа к макросам.


## Совместимость

- API: OpenAI-compatible (подходит для любого API с той же структурой запросов/ответов).
- Реестр: используется `HKEY_CURRENT_USER` — права на запись не требуются.
